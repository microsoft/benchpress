# §4.3 Model-Side Prediction Error Analysis

## Paper mapping

- Section: `sec:reliability_analysis`, model-side paragraph `sec:model_predict`.
- Main table: `tab:model_hypotheses`, generated by `gen_table.py`.
- Main figure: `fig:error_hypotheses_52`, generated by `plot_section52.py`.
- Appendix full grid: `appendix_b_sec4_methods/prediction_error_analysis/model_predictability_factors_full/plot_full.py`.

## Purpose

This directory tests which model-side factors make the fixed Logit Bias ALS point predictor easier or harder to use. H1--H4 are correlational hypotheses over model-level features. H5--H8 are hide-half ablations that modify peer, provider, or observation availability. H9 tests temporal robustness by comparing older vs. more recent training models.

## Inputs

- Canonical score matrix and model metadata from the `benchpress` package.
- Shared hide-half baseline utilities in `_shared.py`.
- H-specific `results.json` files for H1--H9.

## Outputs

- `H1_*`--`H4_*/results.json`: per-model feature/error rows, Spearman tests, and raw predictions.
- `H5_*`--`H8_*/results.json`: paired ablation tests and raw base/treatment predictions.
- `H9_temporal/results.json`: temporal-group definitions, per-k summaries, paired comparisons, and raw predictions.
- `tables/model_hypotheses.tex`: generated table snippet.
- `figures/bp_error_hypotheses_52.{pdf,png}`: main-text composite figure.

## Run

```bash
cd ~/Documents/submission/benchpress/github/experiments/sec4_building_benchpress/error_analysis/model_analysis
bash run.sh
```

For single-hypothesis debugging, run the corresponding `H*/analyze.py` or `H*/ablation.py` from this parent directory. H1--H4 skip rerun when their `results.json` already exists. H5--H9 are deterministic but can be expensive; reuse existing `results.json` unless intentionally rerunning.

## Resume / rerun

To force a clean rerun, delete the relevant `H*/results.json` before running that hypothesis. All expensive outputs store raw predictions with seeds, model indices, benchmark indices, actual scores, and predicted scores so downstream metrics can be recomputed without rerunning the predictor.

## Interpretation

H1--H4 report Spearman correlations between a model feature and prediction error. H5--H8 report paired Wilcoxon tests over one median treatment delta per model. H9 reports paired temporal comparisons across seeds and revealed-benchmark budgets.
